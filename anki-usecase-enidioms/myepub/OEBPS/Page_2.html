1. Mes mots

Spaced repetition ● Répétition espacée

Spaced repetition is a learning technique. La répétition espacée est une technique d'apprentissage.

Note: Un logiciel facilitation la répétition espacée est appelé SRS (Spaced Repetition System).

Flashcard ● Carte mémoire

A flashcard is a set of cards bearing information Une carte mémoire est un ensemble de fiches portant de l'information.
